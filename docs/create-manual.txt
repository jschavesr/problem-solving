
Usage: add_case [Options] Problem

Creates solution folder for problem ${Problem} in the programming language
${Lang}, the created files are:

|- workspace/
|  |- ${Problem}/
|  |  |- main.${Lang}
|  |  |- cases/
|  |  |  |- 0.in
|  |  |  |- 0.out

main.go, 0.in and 0.out will be automatically opened in VSCode.

The ${Problem} variable will be saved in the ./config file.
The default value for ${Lang} is "go". In case it is not given and it is not in
the ./config file.

Options:
  -l,--lang                set the ${Lang} variable in ./config file, and uses
                           creates the solution file with the template for the
                           given language
  -h, --help               show this manual, then exit
